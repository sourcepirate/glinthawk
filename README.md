## Glinthawk

Watches everything on the infra from sky.

![Glinthhawk](assets/glinthawk.png)

## Installation

```
  git clone https://github.com/sourcepirate/glinthawk.git
  cd glinthawk
  cargo build --release
```

## Metrics

* CPU Load Avg
* Memory(RAM) usage
* Process Metrics
* Open connections( TCP, UDP)
